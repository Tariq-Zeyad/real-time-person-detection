# 🚀 Real-Time Object Detection with YOLOv8 + Email Alerts

This project uses **YOLOv8**, **OpenCV**, and **Supervision** to detect people in real-time from a webcam and send an email alert when a person is detected.

---

## 📌 Features

- 🎯 Real-time object detection using YOLOv8
- 👤 Person detection (COCO dataset class 0)
- 📷 Live webcam processing
- 📊 FPS counter display
- 📧 Email alert system when a person is detected
- 🧠 Uses GPU if available (CUDA support)

---

## 🧰 Technologies Used

- Python 3.10+
- [Ultralytics YOLOv8](https://github.com/ultralytics/ultralytics)
- [Supervision](https://github.com/roboflow/supervision)
- OpenCV
- PyTorch
- SMTP (Email alerts)

## ⚠️ Important Notice
If you are using **Gmail**, you MUST NOT use your normal password.

---

# 🔐 What is an App Password?

An App Password is a special 16-character code generated by Google that allows external apps (like this project) to send emails securely.

---

# 🛠️ How to Create an App Password

## 1. Enable 2-Step Verification
Go to:
https://myaccount.google.com/security

- Turn ON **2-Step Verification**
- Complete the setup process

---

## 2. Generate App Password
Go to:
https://myaccount.google.com/apppasswords

Then:

- Select **App → Mail**
- Select **Device → Other (Custom name)**
- Write: `YOLO Project`
- Click **Generate**

---

👉 Copy it **without spaces**

---

## 4. Add it to your project

Open `email_settings.py` and write:

```python
from_email = "your_email@gmail.com"
to_email = "receiver_email@gmail.com"

passwords = "YOUR_APP_PASSWORD_HERE"
